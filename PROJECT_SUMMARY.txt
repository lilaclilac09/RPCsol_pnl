🚀 快速开始（5 分钟）

1️⃣ 验证安装：
   $ bash harness/setup.sh

2️⃣ 配置 API Key：
   $ export HELIUS_API_KEY="your-helius-api-key"

3️⃣ 测试 GBrain 连接：
   $ bun test_gbrain_connection.mjs

4️⃣ 测试单个钱包：
   $ bun harness/router.mjs <wallet-address> sparse

5️⃣ 启动 EvoHarness 进化：
   $ python -m meta.run_iteration \
     --experiment-dir ./runs/rpc-router \
     --iteration 1 \
     --parent-variant baseline \
     --proposer-model haiku \
     --eval-model sonnet \
     --focus-surface periodicChunking \
     --harness-path ./harness

✨ 完成！所有结果自动写入 GBrain。

❓ 常见问题（FAQ）

Q: 如果 HELIUS_API_KEY 过期了怎么办？
A: 重新获取新的 Key，运行：export HELIUS_API_KEY="new-key"

Q: GBrain 数据库占用多少空间？
A: 通常 < 100 MB（除非运行几千次迭代）

Q: 可以在团队中共享 GBrain 吗？
A: 是的，迁移到 Supabase：gbrain migrate --to supabase

Q: 如何清理所有 EvoHarness 迭代？
A: rm -rf runs/

Q: 如何回滚到原始代码？
A: 参考 NEW_FILES_MANIFEST.md 的"恢复到修改前"部分
